----------------NINJA CAT--------------

Ninja Cat is a platform game where you can play as a Ninja Cat and kill zombies! :D

Link to GitHub repository -> https://github.com/Acaree/NinjaCat

-----Input-----

- A/D -> Walk left/right
- Space -> Glide
- W -> Jump
- P -> Attack

-F1 -> Start from first level
-F2 -> Start from the beginning of current level
-F3 -> Change level
-F5 -> Save the current state
-F6 -> Load the previous saved state
-F8 -> Debug UI
-F9 -> Show Colliders
-F10 -> GodMode
-F11 -> Cap to 30/60 FPS

Link to website: https://acaree.github.io/NinjaCat/
